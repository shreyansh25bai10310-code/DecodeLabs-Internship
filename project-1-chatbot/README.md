# Project 1 — Friday Rule-Based AI Chatbot

**DecodeLabs AI Internship | Batch 2026**

## Overview
Friday is a rule-based AI chatbot built using pure Python dictionary logic.
No machine learning — just clean, fast O(1) intent matching.

## Key Concepts
- Dictionary-based knowledge base (O(1) lookup)
- Input sanitization and normalization
- Fallback response handling
- Exit command detection

## How to Run
```bash
python chatbot.py
```

## Sample Interactions
You: hello

Friday: Hey there! I'm Friday. How can I help you today?
You: what is ai

Friday: AI is the simulation of human intelligence by machines using data and algorithms.
You: joke

Friday: Why do programmers prefer dark mode? Because light attracts bugs! 🐛
